# Redux快速上手

Redux 是 JavaScript 状态容器，提供可预测化的状态管理。

# 核心概念：

# Reducer

Reducer是一个方法，他接受一个state和action对象作为参数。state即我们的数据，action指定如何修改数据，reducer将根据action来返回一个新的state。

//示范：
let initCounter = 0;
function(state = initCounter, action) {
  switch(action.type) {
    case 'ADD':
      return state + action.value;
    case 'RESET':
      return initCounter;
    default:
      return state;
  }
}

# Action

Action是一个对象，他描述了如何产生新的数据，即state。一个action至少有一个type属性，他就像一条指令，告诉reducer应该如何操作state，同时action也可包含新数据。

//示范：
{
  type: 'ADD', //指令
  value: 1     //新数据
}

# Store

通过Redux的createStore()方法即可创建一个store对象，该方法接收一个reducer作为参数。 Store对象就是提供当前state的对象，他也负责将action传入reducer进行处理。我们可以注册store的监听者，这样当每次state有改动时，我们便能够及时响应。

store提供了如下方法：

  • 使用 getState() 方法获取 state；
  • 使用 dispatch(action) 方法更新 state；
  • 使用 subscribe(listener) 注册监听器;
  • 使用 subscribe(listener) 返回的函数注销监听器。
//示范：
import { createStore } from 'redux';

const counterReducer = function(state = 0, action) {
  switch(action.type) {
    case 'ADD':
      return state + action.value;
    case 'RESET':
      return 0;
    default:
      return state;
  }
}

const store = createStore(counterReducer);

store.subscribe(() => {
  console.log('counter发生了改变', store.getState())
});

setInterval(() => {
  store.dispatch({
    type: 'ADD',
    value: 1
  });
}, 1000);
